# UC17 - Atualizar Parâmetros Legais

O Analista de RH acessa o menu de parâmetros legais. O sistema lista todos os parâmetros legais cadastrados com seus respectivos valores. O sistema deve exibir para cada parâmetro, no mínimo, o nome do parâmetro, seu valor vigente e a data de início da vigência deste valor.

* O Analista de RH pode atualizar os parâmetros legais acessando a tela de atualização de parâmetros legais.
