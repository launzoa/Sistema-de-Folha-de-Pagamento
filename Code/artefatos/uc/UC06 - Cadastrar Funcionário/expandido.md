
| Nome do Caso de Uso                         | Cadastrar Funcionário                                                                                                                                                                                                                                                                                                                                                                             |

| Ator Principal                              | Administrador ou RH                                                                                                                                                                                                                                                                                                                                                                               |
| Pré-Condições                               | Os dados do funcionário devem estar atualizados e serem válidos para o cadastro.                                                                                                                                                                                                                                                                                                                  |
| Garantias de Sucesso (Pós-Condições)        | O cadastro do funcionário foi realizado com sucesso. Os dados foram cadastrados. Os proventos e descontos foram calculados e cadastrados corretamente.                                                                                                                                                                                                                                            |
| Cenário de Sucesso Principal (Fluxo Básico) | 1. O Administrador ou RH acessa a tela de cadastro de funcionário.<br>2. O Sistema exibe o formulário dos campos de cadastro do funcionário.<br>3. O Administrador ou RH preenche os dados e clica em "Salvar".<br>4. O Sistema executa as regras de validação dos dados do funcionário e salva no Banco de Dados.<br>5. O Sistema emite uma mensagem de confirmação do cadastro do funcionário. |
| Fluxo Alternativo                           | 5a. O Sistema emite uma mensagem de erro de verificação do cpf do funcionário.<br>5a.1 O Sistema cancela o cadastro do funcionário.<br>5a.2 O Sistema retorna para a tela de cadastro do funcionário e retorna para o fluxo básico: 2.                                                                                                                                                            |
