# UC06 - Cadastrar Funcionário

O analista de RH acessa a tela de cadastro de funcionários informando os dados do funcionário para o cadastro: nome, cpf, data de adimissão, cargo, salário bruto, status. O sistema valida as informações do funcionário e cadastra o funcionário.

* Caso o cpf seja inválido ou já cadastrado, o sistema não cadastra o funcionário e exibe uma mensagem de erro.
* Caso as informações sejam válidas, o sistema cadastra o funcionário e retorna para a tela de cadastro de funcionários com uma mensagem de sucesso.
