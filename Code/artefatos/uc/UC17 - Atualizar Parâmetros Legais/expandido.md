
| Nome do Caso de Uso                         | Atualizar Parâmetros Legais                                                                                                                                                                                                                                                                                                                                                                                               |
| ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Ator Principal                              | Administrador ou RH                                                                                                                                                                                                                                                                                                                                                                                                       |
| Pré-Condições                               | Os parâmetros legais devem estar devidamente cadastrados.                                                                                                                                                                                                                                                                                                                                                                 |
| Garantias de Sucesso (Pós-Condições)        | Os parâmetros legais foram atualizados com sucesso.                                                                                                                                                                                                                                                                                                                                                                       |
| Cenário de Sucesso Principal (Fluxo Básico) | 1. O Administrador ou RH acessa a tela de parâmetros legais.<br>2. O Sistema exibe o formulário dos campos de todos os parâmetros legais cadastrados.<br>3. O Administrador ou RH preenche os campos com novos dados.<br>4. O Sistema valida os novos valores atualizados usando as respectivas regras de validação para cada parâmetro.<br>5. O Sistema emite uma mensagem de confirmação da atualização dos parâmetros. |
| Fluxo Alternativo                           | 4.a. O Sistema invalida os novos valores atualizados.<br>4.a.1. O Sistema emite uma mensagem de erro.<br>4.a.2.
